团队介绍

吕克昂国际教育集合了各地的优秀教师，形成了多元的教研团队：沉浸教学界多年的学院派，
初出茅庐的年轻学霸，以及在留中国多年的外教，兼而有之。
但有一点他们是共同的：中教均持业内权威证书，专职外教具有三年以上教学经验与资质。
因此可以说，吕克昂国际教育教研团队的专业化精 细化程度，业界难出其右。
教研团队通力合作，目前已开发独立的精品课程及系统教材，标准化学术化的教案，
完备的学生管理制度，以及高效的助教辅学体系，为吕克昂国际教育的未来可持续发展奠定了坚固的基石。
立校以来，吕克昂教研团队秉承“学术、高效、负责”的理念，
专注托福、雅思、ACT等出国类考试培训和德语、意大利语、法语、日语、西班牙语等小语种培训，帮助万千学子通过留学考试，异国圆梦。
